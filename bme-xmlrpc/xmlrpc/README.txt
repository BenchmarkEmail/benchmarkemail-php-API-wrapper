Benchmark Email XML-RPC Example Code
-------------------------------------------------------------------------------
API Version: 1.0
Tested under: PHP 5+

Requires:
    * PHP 5+
    * PEAR XML-RPC2

===== About =====
This packages contains all the methods support by the API Version 1.0.
This would give you a head start to integrate our code into your own 
website or programs as necessary.

You are required to install PEAR XML-RPC2 for these examples to work.
===== Getting Started =====
1) Open the inc/config.php file and edit at least the Username & Password
    
2) Try running xml-listGet.php from the command line. Something like this:
    # php -f xml-listGet.php
